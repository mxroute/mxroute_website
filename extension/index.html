Alias Manager
Chrome Extension

Create and manage MXroute email aliases instantly. Auto-detects email fields and generates unique aliases for every site you visit.

Features

  • Quick Alias Creation: Click the extension icon or use the floating prompt on any email field to instantly create a new alias
  • Smart Detection: Automatically detects email input fields and offers to create an alias
  • Multiple Formats: Generate aliases based on the website domain, random strings, or custom names
  • Per-Domain Destinations: Route aliases from different domains to different email addresses
  • Alias History: Track all your aliases, see which site they were created for, and manage them from one place
  • Auto-Fill & Copy: New aliases are automatically filled into the form and copied to your clipboard

Download the Extension

Note: This extension is currently pending review on the Chrome Web Store. Once approved, you'll be able to install it directly from there. In the meantime, you can install it manually using the instructions below.

Requirements

MXroute Account: You need an active MXroute email hosting account.

API Key: Generate an API key from your MXroute control panel: management.mxroute.com → Login to Panel → Advanced → API Keys

Browser: Google Chrome or any Chromium-based browser (Edge, Brave, etc.)

Manual Installation Instructions

While the extension is pending Chrome Web Store review, you can install it manually in developer mode:

  1. Download and extract the ZIP file using the download button above. Extract it to a folder on your computer (remember where you put it).
  2. Open Chrome Extensions: Type chrome://extensions in your address bar and press Enter.
  3. Enable Developer Mode: Toggle the "Developer mode" switch in the top-right corner of the page.
  4. Load the extension: Click "Load unpacked" and select the folder where you extracted the ZIP file.
  5. Configure your credentials: The extension settings page will open automatically. Enter your MXroute server hostname, username, and API key.

Getting Your API Key

  1. Go to management.mxroute.com and log in with your MXroute credentials.
  2. Click "Login to Panel" next to your email domain.
  3. In the DirectAdmin panel, navigate to Advanced in the left sidebar.
  4. Click "API Keys" and create a new API key. Copy it immediately as it won't be shown again.

How It Works

The extension uses MXroute's DirectAdmin API to create email forwarders (aliases) on your domains. When you create an alias like amazon@yourdomain.com, any email sent to that address is forwarded to your real email address.

This lets you give every website a unique email address. If one starts receiving spam, you know exactly which site leaked your email, and you can delete that alias without affecting your other accounts.
